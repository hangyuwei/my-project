## CloudBase Mock Backend (Local Admin + Cloud Functions)

### Start
1. `npm install`
2. `node server/index.js`

### Ports
- Default: `3001`
- Override: `PORT=3000 node server/index.js`

### Endpoints
- Admin Console: `http://127.0.0.1:3001/admin`
- Cloud Functions: `POST http://127.0.0.1:3001/cloudfunctions/<name>`
- Database CRUD: `/db/<collection>`
- Storage: `/storage/upload`, `/storage/list`, `/storage/item`
- Logs: `/logs`
- Settings: `/settings`

### Auth
- Admin password: `ADMIN_PASSWORD` (env) or default `admin123`.
- Admin UI requires login when `requireAdminAuth` is enabled.

### Device/Emulator Tips
- For real devices, set `config.cloudBaseUrl` to your LAN IP (not `127.0.0.1`).
- CORS is enabled for local dev.

### Contracts
- Snapshot mock responses: `npm run contracts:gen`
- Diff real API vs mock: `npm run contracts:diff`
